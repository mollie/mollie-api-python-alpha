## list

Retrieve a list of all physical point-of-sale devices.

The results are paginated.

> ðŸ”‘ Access with
>
> [API key](/reference/authentication)
>
> [Access token with **terminals.read**](/reference/authentication)

### Example Usage

```python
import mollie_api_python_beta
from mollie_api_python_beta import Mollie
import os

with Mollie(
    security=mollie_api_python_beta.Security(
        api_key=os.getenv("MOLLIE_API_KEY", ""),
    ),
) as mollie:

    res = mollie.terminals.list(from_="term_7MgL4wea46qkRcoTZjWEH")

    # Handle response
    print(res)

```
